# UI Rules

Use Scandinavian Minimalism.

Rules:

- Large whitespace
- Thin borders
- No heavy shadows
- Monospace labels
- Large headlines
- Rounded cards
- Fast transitions

Animation:

150ms–250ms

Hover:

translateY(-2px)

Charts:

Fade + rise animation

Sidebar:

Active indicator slides

Tables:

Alternating row backgrounds

Modals:

Scale 0.98 → 1
